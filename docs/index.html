ABP 1º Semestre

Sobre o projeto

    Website responsivo desenvolvido para consulta de horário de aula dos cursos da Fatec - Jacarei. O sistema permite que os alunos visualizem suas grades horárias de forma intuitiva e responsiva.

Minha responsabilidade

    No projeto do ABP, atuei principalmente na área de design e front-end. Fui responsável pela criação do protótipo inicial no Figma, definindo a identidade visual e garantindo uma experiência intuitiva para os usuários. Além disso, revisei e ajustei a estética das primeiras páginas desenvolvidas, aprimorando a consistência visual do sistema. Também realizei modificações no mapa do site e assumi a programação da seção da secretaria, contribuindo para o funcionamento das funcionalidades voltadas a esse setor. Já na fase final do projeto, desempenhei atividades adicionais com JavaScript, apoiando na implementação de interações dinâmicas e melhorias na usabilidade.

Funcionalidades

  • Design responsivo para mobile e desktop
  • Consulta de horários por turma ou professor
  • Interface intuitiva e moderna
  • Sistema de busca integrado
